# Code availability

Name of the code: FD-CMA: frequency-domain cross-modal alignment for GPR--ERT dam defect detection.

Contact: Wenqiang Ding, wq-ding@outlook.com.

Program language: Python.

Software required: Python 3.9 or later; PyTorch; CUDA-compatible GPU environment is recommended for detector training; additional Python packages are listed in `requirements.txt`.

Hardware requirements: GPU acceleration is recommended for model training. CPU execution is possible for feature construction and lightweight preprocessing, but is not recommended for full detector training.

The source codes are available for downloading at the link: https://github.com/bimyoukyori/FD-CMA.
